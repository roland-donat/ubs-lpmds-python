2. Exercice : Conversion des durées

Thème: Variables, opérations arithmétiques, fonction, tuple, dictionnaire

Supposons que nous disposons d'une variable dont la valeur est un nombre de secondes : 

nombre_secondes = 3569

Travail à réaliser :

  1. Écrire un code Python permettant de convertir ce nombre de secondes en jours, heures, minutes et secondes et d'afficher le résultat. Vous devez bien sûr utiliser la variable nombre_secondes dans vos calculs.

    Aides : Pensez aux opérateurs // et % et à la fonction print.

    Solution : 

    nombre_minutes = nombre_secondes // 60
nombre_secondes_restantes = nombre_secondes % 60
nombre_heures = nombre_minutes // 60
nombre_minutes_restantes = nombre_minutes % 60
print("nombre d'heures = " + str(nombre_heures))
print("nombre de minutes = " + str(nombre_minutes_restantes))
print("nombre de secondes = " + str(nombre_secondes_restantes))

  2. Encapsuler votre code dans une fonction prenant en paramètre un nombre de secondes et qui retourne un tuple de taille 4 de la forme suivante (<nb jours>, <nb heures>, <nb_minutes>, <nb_secondes>). La structure de la fonction à constuire est : 

    def convertisseur_secondes(nombre_secondes):
    # Code à écrire
    return nb_jours, nb_heures, nb_minutes, nb_secondes

    Voici quelques exemples de résultats attendus : 

    convertisseur_secondes(nombre_secondes=10)
(0, 0, 0, 10)
convertisseur_secondes(nombre_secondes=100)
(0, 0, 1, 40)
convertisseur_secondes(nombre_secondes=1000)
(0, 0, 16, 40)
convertisseur_secondes(nombre_secondes=10000)
(0, 2, 46, 40)
convertisseur_secondes(nombre_secondes=100000)
(1, 3, 46, 40)

    Solution : 

    def convertisseur_secondes(nombre_secondes):
    nombre_minutes = nombre_secondes // 60
    nb_secondes = nombre_secondes % 60
    nombre_heures = nombre_minutes // 60
    nb_minutes = nombre_minutes % 60
    nb_jours = nombre_heures // 24
    nb_heures = nombre_heures % 24

    return nb_jours, nb_heures, nb_minutes, nb_secondes

  3. Modifier la fonction précédente pour que le résultat soit fourni sous la forme d'un dictionnaire de la forme {"jour": <nb jours>, "heures": <nb heures>, "minutes": <nb_minutes>, "secondes": <nb_secondes>}

    Solution : 

    def convertisseur_secondes_bis(nombre_secondes):
    nb_jours, nb_heures, nb_minutes, nb_secondes = convertisseur_secondes(nombre_secondes)
    return {"jour": nb_jours,
            "heures": nb_heures,
            "minutes": nb_minutes,
            "secondes": nb_secondes}

3. Exercice : indexation des chaînes de caractères

Thème: Chaînes de caractères, indexation

Considérons la chaîne de caractères suivante : 

dicton = "Le jour où j'ai découvert Python, j'ai abandonné R !"

Travail à réaliser :

  1. Utiliser l'indexation pour extraire la chaîne j'ai abandonné R ! de la chaîne dicton.

    Solution : 

    dicton[34:]

  2. Faire de même en utilisant une indexation négative.

    Solution : 

    dicton[-18:]

  3. Utiliser la méthode split pour découper la chaîne dicton en utilisant le séparateur " " (un espace) en liste de termes : 

    dicton.split(" ")

       ['Le', 'jour', 'où', "j'ai", 'découvert', 'Python,', "j'ai", 'abandonné', 'R', '!']

  4. Écrire une fonction calcule_nb_mots qui prend en argument une chaîne de caractères et renvoie le nombre de mots d'une phrase. On considère qu'un mot est une chaîne de caractères ne contenant ni ponctuation, ni espace. Voici quelques exemples de résultats attendus : 

    calcule_nb_mots(dicton)
9
calcule_nb_mots("  Il fait beau  ce matin")
5
calcule_nb_mots("Voulez-vous un café ?")
4

    Penser à utiliser les méthodes suivantes <chaine>.split ; <chaine>.replace ; <chaine>.strip ; len(<chaine>).

    Solution : 

    def calcule_nb_mots(chaine):
    # Nettoyage
    # Suppresion des espaces multiples
    chaine_clean = chaine.replace("  ", "")
    # Suppresion de la ponctuation
    ponc_list = ",;.!?'-"
    for ponc in ponc_list:
        chaine_clean = chaine_clean.replace(ponc, "")
        
    liste_mots = chaine_clean.strip().split(" ")
    return len(liste_mots)

5. Exercice : The Classic Fizz Buzz Problem

Thème : Structures de contrôle.

Le Fizz Buzz est un problème classique en informatique. Ce dernier est souvent utilisé pour tester des candidats lors d'un entretien d'embauche.

La légende raconte que de nombreuses personnes échouent à ce test lors des entretiens.

Travail à réaliser : Écrire un programme qui affiche les nombres de 1 à 100 mais :

  • pour les multiples de 3, affiche Fizz à la place du nombre.
  • pour les multiples de 5, affiche Buzz à la place du nombre.
  • Pour les multiples de 3 et 5, affiche FizzBuzz à la place du nombre.

Aide : Penser à l'opérateur %.

6. Exercice : Traitement de données textuelles

Source : exercice adapté à partir de PythonCert.

Thème : fichiers, structures de données, fonctions.

Télécharger le fichier students.txt dans votre espace de travail.

Ce fichier contient une liste de personnes avec, pour chacune d'elles, les langages de programmations maîtrisés.

Le format du fichier est le suivant :

  • la première ligne du fichier est une entête rappelant le format ;
  • chaque ligne est construite comme suit : Lastname, Firstname: Nickname, languages ;
  • les éléments Lastname et Firstname sont toujours présents et sont séparés par une virgule et un espace ;
  • après le Firstname, on trouve le caractère : ;
  • le Nickname commence toujours par une majuscule ;
  • les langages commencent toujours par une minuscule ;
  • chaque personne peut avoir un Nickname ou aucun ;
  • chaque personne peut maîtriser zéro ou un ou plusieurs langages.

L'objectif général de l'exercice est de parcourir le fichier afin d'extraire les informations de chaque personne en les organisant dans une structure de données adéquates.

Travail à réaliser :

  1. Ouvrir le fichier en lecture. Aide : fonction open et structure with ... as.
  2. Extraire pour chaque personne, son nom, prénom, surnom (si présent) et liste des langages (si présente). Aide : méthodes .readlines .split, .strip.
  3. Stocker ces éléments dans une structure adaptée. Nous appellerons cette structure data_struct dans la suite.
  4. À partir de la structure data_struct, déterminer la liste des langages maîtrisés par les individus présents dans le fichier.
  5. À partir de la structure data_struct, calculer le nombre de personnes maîtrisant chaque langage.
  6. Créer deux fonctions prenant en entrée la structure data_struct et retournant respectivement le résultat des deux questions précédentes.
  7. Créer une fonction prenant en entrée le chemin d'un fichier respectant le format du fichier student.txt et retournant la structure data_struct correspondante.

7. Exercice : Notion de compréhension

Considérons la liste suivante : 

fruits = ['mangue', 'kiwi', 'framboise', 'noix de coco', 
          'goyave', 'pamplemousse', 'orange sanguine']

Travail à réaliser : En utilisant le mécanisme de compréhension sur la liste fruits :

  1. Construire une liste contenant les noms de fruits en majuscules.
  2. Construire une liste contenant les fruits ayant un nom de plus de 10 caractères.
  3. Construire une liste contenant les fruits à noms composés.
  4. Construire un dictionnaire ayant pour clés les fruits et pour valeurs les nombres de caractères des fruits correspondants.
  5. Construire un dictionnaire ayant pour clés les fruits et pour valeurs les positions de ces fruits dans la liste. Aide : utiliser la fonction enumerate.
