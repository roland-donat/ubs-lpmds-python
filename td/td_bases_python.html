4. Exercice : Formatage des chaînes de caractères

Thème : Format des chaînes de caractères.

Pour présenter des résultats, il souvent utile d'inclure des résultats dans des chaînes de caractères avec le format adéquat.

Pour ce faire, les chaînes de caractères Python dispose de la méthode .format. Cette méthode s'utilise de la manière suivante : 

"La {texte_1} format c'est {texte_2}".format(texte_1="méthode", texte_2="super")

Pour les nombres, vous pouvez également préciser le format affiché. Par exemple : 

"Un pourcentage {x:.2%} avec deux chiffres après la virgule".format(x=0.4535)
"Un nombre entier sur 3 caractères {x:03} complété avec des zéros".format(x=34)

Considérons les variables suivantes : 

num = 2
ts = 52312.5764
owner = 655
occ_pct = 0.001
size = 154232332

Consultez la documentation officielle, en particulier la section 6.1.3 pour connaître les différentes fonctionnalités de la méthode .format.

Travail à réaliser :

  1. Utiliser la méthode .format afin de produire la chaîne suivante : 

    "fichier_002 :  52312.58, 655,   0.1%, 1.54e+08"
  2. Même question en utilisant la technique des f-string à la place de la méthode .format.

6. Exercice : Notion de compréhension

Thème : Listes, dictionnaires, syntaxe de compréhension

Le principe de compréhension en Python permet de construire des liste ou des dictionnaires "à la volée" à partir d'autres listes ou dictionnaires.

Considérons la liste suivante : 

fruits = ['mangue', 'kiwi', 'framboise', 'noix de coco', 
          'goyave', 'pamplemousse', 'orange sanguine']

Travail à réaliser : En utilisant le mécanisme de compréhension sur la liste fruits :

  1. Construire une liste contenant les noms de fruits en majuscules. 

    [fr.upper() for fr in fruits]
  2. Construire une liste contenant les fruits ayant un nom de plus de 10 caractères.
  3. Construire une liste contenant les fruits à noms composés.
  4. Construire un dictionnaire ayant pour clés les fruits et pour valeurs les nombres de caractères des fruits correspondants.

  5. Construire un dictionnaire ayant pour clés les fruits et pour valeurs les positions de ces fruits dans la liste.

    Aide : utiliser la fonction enumerate.

Source : exercice adapté à partir de PythonCert.

Thème : fichiers, structures de données, fonctions.

Télécharger le fichier students.txt dans votre espace de travail.

Ce fichier contient une liste de personnes avec, pour chacune d'elles, les langages de programmations maîtrisés.

Le format du fichier est le suivant :

  • la première ligne du fichier est une entête rappelant le format ;
  • chaque ligne est construite comme suit : Lastname, Firstname: Nickname, languages ;
  • les éléments Lastname et Firstname sont toujours présents et sont séparés par une virgule et un espace ;
  • après le Firstname, on trouve le caractère : ;
  • le Nickname commence toujours par une majuscule ;
  • les langages commencent toujours par une minuscule ;
  • chaque personne peut avoir un Nickname ou aucun ;
  • chaque personne peut maîtriser zéro ou un ou plusieurs langages.

L'objectif général de l'exercice est de parcourir le fichier afin d'extraire les informations de chaque personne en les organisant dans une structure de données adéquates.

    En pratique, il est plus simple d'utiliser la méthode .readlines qui récupère d'un seul coup la liste des lignes du fichier :
